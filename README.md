# Deutsche eMark [DEM] #

### Version 1.2.0: support LevelDB 
(you must reload DB, if you don´t want, then use version 1.1.0 (BerkleyDB) [from here](https://github.com/emarkproject/DEM/releases ))

- SHA256D, Proof of Work + Proof of Stake
- max 210 Million Proof of Work Coins
- 3.8% annual stake per year, 30 Days Coin-Age required
- 50 coins per block
- 2 minute block targets
- 100 blocks to coin maturation
- 140 Character Transaction Messaging
